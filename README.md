# 🚀 FastAPI CRUD Web App

✨ A full-stack web application built using **FastAPI**, **PostgreSQL**, and a modern **HTML/CSS/JS UI**.

---

## 🌟 Features

✅ Create User
✅ Read User
✅ Update User
✅ Delete User
✅ Responsive Dashboard UI
✅ 3D Animated Background 🎨
✅ Real-time Data from PostgreSQL

---

## 🛠️ Tech Stack

* ⚡ FastAPI (Backend)
* 🗄️ PostgreSQL (Database)
* 🧠 SQLAlchemy (ORM)
* 🎨 HTML, CSS, JavaScript (Frontend)
* ☁️ Render (Deployment)

 🌐 Live Demo
